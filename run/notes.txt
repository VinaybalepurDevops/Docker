Run is used to run system related queries such as apt-get update. CMD is used to run fixed queries. This cannot be changed and the values of CMD command remains fixed. Entry point is also used to run service level commands like CMD, the difference is the values for this can change, where as for CMD the values are fixed. In the ex, Entry point curls google.com by default as the CMD has the value google.com, we can also pass other URLs to the file during the run commands. Such as run -it <ImageID> newURL
